Publications

DeRide: An Incentivized Peer-to-Peer Ride Sharing Service using Permissioned Blockchain Networks

Shailesh Kumar Sharma Sandip Chakraborty Balaji Palanisamy Shamik Sural

IEEE International Conference on Web Services (ICWS) 2025

Under Review

Most of the existing ride sharing services operate on centralized platforms that control pricing, monopolize user data and lack transparency, leading to unfair market dominance. In this paper, we propose DeRide, a decentralized ride sharing scheme built on top of a permissioned blockchain, namely Hyperledger Fabric (HLF), which enables direct peer-to-peer interactions. It promotes transparent and equitable distribution of benefits, thus addressing the shortcomings of centralized systems. DeRide works with a non-monetary token-based model, where users earn tokens by offering to drive but need to spend the same for enjoying a ride. Such a non-monetary incentivization mechanism ensures fair participation from all users. Given a set of drivers and riders, along with their travel locations, assigning maximum number of riders to drivers is framed as as an optimization problem under the constraints that each driver has a certain number of available seats, and would be willing to deviate only up to a maximum distance from its intended travel path for picking up riders. The riders must have enough tokens to their credit for enjoying the ride. The problem being NP-complete, we propose a heuristic solution for the same. The algorithm has been implemented as HLF chaincodes to support verifiability of token accounting and is integrated with Google Map for a Point-and-Select location service. An extensive set of experiments show the viability of DeRide in a realistic scenario.
